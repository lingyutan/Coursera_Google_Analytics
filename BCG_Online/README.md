# The BCG Open-Access Data Science &amp; Advanced Analytics Virtual Experience Program

There are 5 tasks to complete in this virtual program:

1. Business Understanding & Problem Framing
2. Exploratory Data Analysis & Data Cleaning
3. Feature Engineering
4. Modeling & Evaluation
5. Presentation

Task descriptions can be found in each subfolder. My first version of solutions is written in R, and the sample solutions from [here](https://github.com/srgarcess/BCG) are in Python.
